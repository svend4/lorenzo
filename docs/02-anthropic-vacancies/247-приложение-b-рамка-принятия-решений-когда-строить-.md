## Приложение B: Рамка принятия решений — когда строить Тип 1 первым

Для организации, решающей, строить ли
Профессионального Коллегу-Агента или другой тип:

**Выбирайте Тип 1 (Профессиональный Коллега-Агент), когда:**
- Профессия имеет хорошо определённые
  стандартизованные практики
- Практикующие сталкиваются с существенной
  рутинной нагрузкой
- Профессиональный контекст кодируется в базу
  знаний
- Нет сильного требования внешней коммуникации
- Регулятивная среда для профессии яснее, чем
  для AI

**Выбирайте Тип 0 (Персональный AI-помощник), когда:**
- Случай использования общего назначения, не
  профессионально-специфичный
- У пользователя нет конкретного
  профессионального контекста для встраивания
- Массовый потребительский продукт

**Выбирайте Тип 2 (Институциональный Агент), когда:**
- Институция имеет конкретные потребности
  интеграции данных/процессов
- Институциональные правила центральны для
  поставки ценности
- Кастомизация по институции ожидается и
  принимается

**Выбирайте Тип 3 (Агент-Сотрудник), когда:**
- Рутинная задача может быть чисто делегирована
- Рамка ответственности позволяет автономное
  действие
- Качество рутинных результатов проверяемо

**Выбирайте Тип 4 (Представительский Агент), когда:**
- Индивидуальные пользователи нуждаются в
  навигации по рынку
- Личный контекст богатый и уникальный
- Долгосрочное представительство карьерной
  траектории ценно
- Инфраструктура доверия зрелая

Для большинства финансируемых фондом
развёртываний, обслуживающих профессиональные
группы, Тип 1 — правильная отправная точка.
