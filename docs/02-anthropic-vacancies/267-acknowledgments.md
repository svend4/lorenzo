## Acknowledgments

The Composite Skills Agent concept emerged from dialogue with 
Claude (Anthropic) on April 26, 2026. The crystallizing 
contribution was the user's observation that the binary view 
between Professional Colleague Agent (Type 1) and Representative 
Agent (Type 4) missed the most common middle case — practitioners 
whose distinctive value comes from combining individually general 
specializations.

The metaphor of the Indian yoga master who studied with twenty 
teachers, drawn from a journalist's interview the user 
remembered, provides the central illustration that makes the 
pattern clear.

The user retained editorial authority. All claims, errors, and 
specific architectural proposals are author's responsibility.

This paper is the seventh in the sequence developed across this 
extended collaboration:
1. Nautilus Portal Protocol v1.1
2. Review Methodology v1.0
3. Double-Triangle Architecture v1.0
4. Open Knowledge Work Foundation Concept Document v1.0
5. Representative Agent Layer v1.0
6. Professional Colleague Agents v1.0
7. Composite Skills Agent v1.0 (this paper)

Together these papers form an architectural vision for 
AI-supported knowledge work that respects how mastery develops, 
how professions function, and how human dignity is preserved 
when AI augments human practice.
