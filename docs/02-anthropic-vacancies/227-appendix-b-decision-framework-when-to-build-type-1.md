## Appendix B: Decision Framework — When to Build Type 1 First

For an organization deciding whether to build a Professional
Colleague Agent vs. another type:

**Choose Type 1 (Professional Colleague Agent) when:**
- Profession has well-defined standardized practices
- Practitioners face significant routine burden
- Professional context is encodable in knowledge base
- No strong external communication requirement
- Regulatory environment for profession is clearer than for AI

**Choose Type 0 (Personal AI Assistant) when:**
- Use case is general-purpose, not profession-specific
- User has no specific professional context to embed
- Mass-market consumer product

**Choose Type 2 (Institutional Agent) when:**
- Institution has specific data/process integration needs
- Institutional rules are central to value delivery
- Per-institution customization is expected and accepted

**Choose Type 3 (Employee Agent) when:**
- Routine task can be cleanly delegated
- Liability framework permits autonomous action
- Quality of routine outputs is verifiable

**Choose Type 4 (Representative Agent) when:**
- Individual users need market navigation
- Personal context is rich and unique
- Long-term career-arc representation is valuable
- Trust infrastructure is mature

For most foundation-funded deployments serving professional
populations, Type 1 is the right starting point.
