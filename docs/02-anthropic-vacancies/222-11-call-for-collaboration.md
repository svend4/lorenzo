# 11. Call for Collaboration

## 11. Call for Collaboration

We invite specific kinds of collaboration:

### 11.1. For Professional Practitioners

**Especially in social law, education, healthcare, social
work**: Tell us what burdens you most. What routine work
absorbs your time? What would you delegate to an AI colleague?
What would you never delegate? Pilot participation invited.

### 11.2. For Domain Experts (Curators)

Layer A construction requires experts willing to encode
professional knowledge into structured form. This is detailed,
careful work. We need curators willing to invest time —
preferably compensated.

### 11.3. For Software Engineers

Build Layer B and C. Open-source contributions to reference
implementations welcome. Particularly engineers with
experience in:
- LLM application architecture
- Multi-language support
- Accessibility
- Privacy-preserving computation

### 11.4. For Foundation Funders

Foundation-funded Professional Colleague Agents are a
high-leverage mission opportunity. €0.5M-3M can deploy
agent for a profession of 10,000-100,000 practitioners.
Population impact through practitioner amplification is
substantial.

### 11.5. For Existing Professional Tool Vendors

Many existing tool vendors are evolving toward Professional
Colleague Agent capability. Open standards collaboration
accelerates field. Collaboration on Layer A standards
particularly valuable.

### 11.6. For Researchers

Empirical studies of Professional Colleague Agent deployment
needed. Effects on practice quality, profession evolution,
practitioner satisfaction, client outcomes. Several years
of deployment data now becoming available («Обучай», early
medical AI deployments).

### 11.7. For Policy Stakeholders

Regulatory frameworks for Professional Colleague Agents are
nascent. Liability, professional licensure interaction,
quality standards — all need development. Engagement with
practitioners during framework development valuable.
