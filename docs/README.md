# docs/ — тематический индекс

Каждый каталог здесь — это отдельная большая тема. Внутри каждой темы — подпапки по подтемам, и в каждой подпапке — маленькие документы, по одному на узкий вопрос.

## Темы

| Папка | О чём |
|---|---|
| [`svyazi-2-0/`](svyazi-2-0/) | Главное исследование: software-first конструктор для Svyazi 2.0, ансамбли, контракты, MVP, безопасность, контактная стратегия. Источник: `deep-research-report (1)/(2)/(3)/(4).md`. |
| [`anthropic-vacancies/`](anthropic-vacancies/) | Анализ вакансий Anthropic по кластерам. Источник: MHTML-снимок диалога с claude.ai. |

| [`ai-collaborations/`](ai-collaborations/) | Поиск коллабораций AI-проектов. Источник: MHTML-снимок диалога с claude.ai. |
| [`habr-unique-projects/`](habr-unique-projects/) | Поиск уникальных проектов на Хабре для совместной разработки. Источник: MHTML-снимок диалога с claude.ai. |
| [`glossary/`](glossary/) | Кросс-ссылочный словарь компонентов, авторов и понятий — единая точка входа для поиска всех упоминаний одного проекта. |

## Принципы организации

- Каждая папка содержит свой `README.md` — индекс подтем.
- Маленькие документы (1 тема — 1 файл).
- Никакой исходный документ не удалён; здесь только перенесённое и разбитое содержимое.
- Цитаты `citeturn…` сохраняются как есть, чтобы не терять связь с исходным research-trace.
