> Источник: MHTML‑снимок `Поиск уникальных проектов на Хабре для совместной разработки - Claude` (корень репозитория).

LiteLLM маршрутизирует запросы между Opus, локальным Qwen3:8B и DeepSeek-V3.2 — каждый запрос идёт в самую дешёвую достаточную модель;

скилл-каталог твоих 87+ skills устанавливается в Claude Code одной командой — каждый skill знает свою роль;

voice-input через Handy ловит твои мысли по ходу дня;

tmux-деревня из 5-6 агентов с heterogeneous моделями параллельно ведёт несколько дел;

vault в Obsidian + RAG над corpus-ом юридических документов с pdfplumber-подсветкой — единый источник правды.

Получается: один человек физически способен вести 30-50 дел Sozialrecht параллельно с качеством, которое в офисе с пятью юристами. Это не теория — суммарно все компоненты уже работают по отдельности у разных авторов на Хабре.
