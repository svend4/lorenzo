Это два MHTML‑снимка одного и того же диалога с claude.ai (вторая копия — практически идентичная). Не удалены, как просил пользователь.

Тема диалога — как находить уникальные проекты на Хабре, которые подходят для совместной разработки (часть мысли, продолжающаяся в `deep-research-report (1)/(3)`).

## Подпапки

- [`search-strategy/`](search-strategy/) — как искать проекты на Хабре.
- [`evaluation/`](evaluation/) — как оценивать уникальность и зрелость найденного.

## Подтемы для дальнейшего разбиения

| Папка / файл | О чём |
|---|---|
| `search-strategy/queries.md` | Какие запросы и хабы давать в поиск |
| `search-strategy/authors.md` | Как двигаться по авторам (а не только по темам) |
| `search-strategy/comments.md` | Что искать в комментариях, а не в самой статье |
| `evaluation/uniqueness.md` | Что считать «уникальным», а не просто «новым» |
| `evaluation/maturity.md` | Шкала зрелости (эксперимент / прототип / активный OSS / закрытый прототип) |
| `evaluation/license.md` | Как читать лицензию и почему важна BSL/MIT/Apache развилка |
| `evaluation/integration-fit.md` | Как оценивать «совместимость без переписи стека» |
| `case-studies.md` | Конкретные кейсы найденных проектов |
