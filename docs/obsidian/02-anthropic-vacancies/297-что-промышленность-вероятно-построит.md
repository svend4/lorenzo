## Что промышленность вероятно построит

Независимо от того, что делает автор,
промышленность будет двигаться к
инфраструктуре Слоя B в течение 1-3 лет.
Пробел слишком очевиден, чтобы оставаться
незаполненным.

Вероятные направления:

**Расширение Anthropic Projects.** Projects
вероятно добавят некоторый тред,
версионирование документов и более богатую
организацию. Anthropic позиционирован, чтобы
делать это хорошо, потому что они тесно
контролируют AI-интеграцию. Ожидайте
инкрементальные добавления к функциональности
Projects через 2026-2027 годы.

**GitHub для идей.** GitHub может построить
что-то параллельное GitHub для кода —
интеллектуальный продукт рабочего пространства.
У них есть инфраструктура, пользовательская
база и бренд. Риск: культура кода будет
сохраняться.

**Углубление Notion AI.** Notion может
расширить Notion AI от помощи в генерации до
подлинного сотрудничества. У них есть
структурная способность (базы данных,
связывание, комментарии). Риск: сложность
Notion создаёт барьер для входа.

**Специализированный стартап.** Какая-то новая
компания может построить Слой B специально.
Это наиболее вероятный путь к подлинно новой
инфраструктуре, но самый рискованный.

**Проект open source сообщества.**
Исследователи и интеллектуальные работники
могут построить свой собственный.
Hypothes.is для аннотаций существует; мог бы
быть расширен. Discourse плюс AI-плагины.
Различные комбинации.

Мы не должны ждать какого-либо конкретного
исхода. Мы должны использовать то, что
существует, и адаптироваться по мере
появления лучшей инфраструктуры.
