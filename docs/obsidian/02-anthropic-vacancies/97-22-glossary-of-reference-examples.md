## 22. Glossary of Reference Examples

В качестве иллюстраций используется экосистема `svend4` с 7 Repos:

- **[[01-интегральный-анализ-профиля-svend4|info1]]** (Level 1) — 74+ Markdown-документа с α-уровнями 
  (-4..+4), методологический угол
- **[[01-интегральный-анализ-профиля-svend4|pro2]]** (Level 3) — Q6-концепты, граф знаний, bidir-цикл, 
  семантический угол
- **meta** (Level 1) — 256 CA-правил + 64 гексаграммы, символьный 
  угол
- **data2** (Level 1) — 310+ томов ЕТД (Единая Теория Движения)
- **data7** (Level 2) — граф знаний, K₀→K∞ трансформация
- **infosystems** (Level 2, домен [[01-интегральный-анализ-профиля-svend4|pro2]]) — архитектура ИС
- **ai_agents** (Level 2, домен [[01-интегральный-анализ-профиля-svend4|pro2]]) — ИИ-агенты и мультиагентные 
  системы

Плюс 5 "расширенных" адаптеров (не в registry, но useable):

- **ObsidianAdapter** — локальный Obsidian vault ([[wikilinks]])
- **ArxivAdapter** — arXiv API с 12h cache
- **GitHubTopicAdapter** — discovery по [[03-component-catalog|GitHub]] топику
- **JSONLAdapter** — JSON Lines files
- **[[141-4-nautilus-portal-as-reference-substrate|AutoAdapter]]** — любой repo с nautilus.json в корне
