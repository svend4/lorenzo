## Ссылки

Ссылки для этого документа более скудные, чем
для прикладных статей, потому что документ
исследовательский, а не специфицирующий.
Ключевая релевантная работа включает:

### Тред и Аннотация

- Проект Hypothesis (https://web.hypothes.is/)
  — open source веб-аннотация
- Программное обеспечение форума Discourse
  (https://www.discourse.org/) — современный
  тредированный форум
- Эксперименты Federated Wiki Уорда
  Каннингхэма — альтернативные модели для
  совместного письма

### Управление Знаниями

- Sönke Ahrens (2017). *Как делать умные
  заметки*. Cambridge University Press.
  (Метод Zettelkasten, основополагающий для
  современного управления личными знаниями.)
- Andy Matuschak (продолжающееся). Рабочие
  заметки об инструментах для мысли.

### Инфраструктура AI-Сотрудничества

- Документация Anthropic Projects (текущая)
- Microsoft Copilot Studio (другой подход к
  аналогичной проблеме)
- Различные мульти-агент рамки (LangChain,
  CrewAI, AutoGen)
