## References

The references for this document are sparser than for the 
applied papers, because the document is exploratory rather 
than specifying. Key relevant work includes:

### Threading and Annotation

- Hypothesis project (https://web.hypothes.is/) — open 
  source web annotation
- Discourse forum software (https://www.discourse.org/) — 
  modern threaded forum
- Federated Wiki experiments by Ward Cunningham — alternative 
  models for collaborative writing

### Knowledge Management

- Sönke Ahrens (2017). *How to Take Smart Notes*. Cambridge 
  University Press. (Zettelkasten method, foundational for 
  modern personal knowledge management.)
- Andy Matuschak (ongoing). Working notes on tools for thought.

### AI Collaboration Infrastructure

- Anthropic Projects documentation (current)
- Microsoft Copilot Studio (different approach to similar 
  problem)
- Various multi-agent frameworks (LangChain, CrewAI, AutoGen)
