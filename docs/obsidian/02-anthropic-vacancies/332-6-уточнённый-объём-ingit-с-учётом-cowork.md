## 6. Уточнённый объём InGit с учётом Cowork

Оригинальная дорожная карта InGit (10-16
месяцев до v1.0) отражала амбицию полного
автономного продукта. С Cowork как дополнением,
объём может драматически сократиться.

### 6.1. Новая Фаза 1 (Месяцы 1-3): Подложка MVP

**Цель**: InGit становится полезным как
подложка Cowork.

**Поставки**:
- Конвенции структуры папок (уже в репозитории)
- YAML схемы метаданных для задач, документов,
  wiki страниц
- Pre-commit хуки для валидации
- Базовое шифрование для чувствительных папок
  (используя age)
- Документация конвенций
- Примеры и шаблоны

**Удалено из оригинальной Фазы 1**:
- Desktop GUI (отложено до «возможно никогда»,
  учитывая Cowork)
- Слой базы данных (SQLite по-прежнему
  полезен, но вторичен)
- Web UI (Cowork — это UI)

**Усилие**: Достижимо за 3 месяца
индивидуальным разработчиком, работающим
неполный рабочий день.

### 6.2. Новая Фаза 2 (Месяцы 3-6): MCP Сервер

**Цель**: Операции InGit доступны Cowork через
MCP.

**Поставки**:
- Python пакет `[[306-with-anthropic-s-cowork-platform|ingit]]-mcp-server`
- 15-20 инструментов, покрывающих основные
  операции
- Open source релиз
- Документация интеграции
- Примеры для Cowork Projects

**Усилие**: 2-3 месяца для соло-разработчика.

### 6.3. Новая Фаза 3 (Месяцы 6-12): Принятие и Уточнение

**Цель**: Реальное использование стимулирует
улучшения.

**Поставки**:
- Инструменты миграции (Notion, Obsidian,
  [[03-component-catalog|GitHub]])
- Уточнения на основе обратной связи
  пользователей
- Расширение документации
- Построение сообщества

**Усилие**: Зависит от скорости принятия.

### 6.4. Общий Сокращённый Объём

Оригинальный план: 10-16 месяцев до v1.0 с
существенными функциями.
Пересмотренный план: 12 месяцев до зрелой
подложки + MCP интеграция.

**Удалённый объём**:
- Desktop GUI: -3 месяца усилий
- Web UI: -2 месяца усилий
- Мобильное приложение: -2 месяца усилий
  (было неявным)
- Инфраструктура агента: -3 месяца усилий
  (Cowork обеспечивает)
- Независимая экосистема коннекторов: -2
  месяца усилий

**Чистое сокращение усилий**: ~12 месяцев
работы удалено.
**Чистая временная линия**: От 10-16 месяцев
до примерно 12 месяцев, с гораздо более
высоким качеством результата.

Это ценность стратегического позиционирования:
меньше работы для лучших результатов.
