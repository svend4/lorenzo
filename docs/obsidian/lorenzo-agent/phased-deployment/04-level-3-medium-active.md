УРОВЕНЬ 3 — Средний (Lorenzo Active)

Что это: Lorenzo получает реальные autonomous capabilities в защищённых рамках.

Возможности:

Автоматический мониторинг Хабра, ArXiv, GitHub, новостей (ежедневно)

Knowledge base в proper структуре (CardIndex pattern от Свяжи)

Synthesis proposals генерируются автоматически еженедельно

Outreach drafts генерируются автоматически (всё ещё Макс review/send)

Blog posts генерируются для Макса (он approves/edits/publishes)

Conversation handling — Lorenzo может вести email threads (с человеком в цикле)

Реализуется: Месяцы 3-6

Что нужно:

Существенный technical work (10-20 hours/week Макса или developer)

Claude API credits регулярно (~€100-300/месяц)

Возможно local LLM backup (Ollama)

Proper database (SQLite или PostgreSQL)

MCP servers для Lorenzo's tool access

Monitoring dashboard для Макса

Стоимость: ~€500-1000 setup + €200-500/месяц

Ограничения:

Всё ещё под Макса approval для всех external actions

Synthesis quality зависит от model capabilities

Нет реальной autonomy в decisions

Ценность: Lorenzo делает реальную работу, существенно усиливает Макса.
