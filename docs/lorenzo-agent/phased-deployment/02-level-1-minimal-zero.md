# Уровень 1 — Минимальный (Lorenzo Zero)

> Источник: MHTML‑снимок `Вакансии в Anthropic по кластерам - Claude` (корень репозитория). Раздел диалога — пятиуровневая phased deployment Lorenzo (от ручного режима к полноценному network).

УРОВЕНЬ 1 — Минимальный (Lorenzo Zero)

Что это: Базовая персистентность через Claude Projects + ручной workflow. Lorenzo существует как конфигурация Project, но всё ещё через ручные триггеры.

Возможности:

Claude Project «Lorenzo Catalyst Agent» с системным промптом, определяющим персону

Project knowledge: накопленная база (CardIndex авторов, проектов, паттернов)

Ручной workflow: Макс открывает Project, говорит «найди что нового в области X», Claude в персоне Lorenzo выполняет

Документы persistified: все находки сохраняются в Project knowledge

Реализуется: Эта неделя — следующая неделя

Что нужно:

Создать Claude Project «Lorenzo Catalyst Agent»

Написать системный промпт, определяющий персону Lorenzo (использует Формулировку B)

Загрузить существующие 9 документов как context

Загрузить findings из текущих сессий (HMP, Свяжи, Анастасия, Виталий и т.д.)

Использовать активно в течение недели для тестирования

Стоимость: €0 дополнительно (если Claude Pro подписка уже есть)

Ограничения:

Всё ещё triggered Максом (не автономный)

Нет внешних действий (нет email, нет GitHub posts)

Память Project ограничена

Не публичное лицо ещё

Ценность: Persistified Lorenzo persona, готов к scaling.
