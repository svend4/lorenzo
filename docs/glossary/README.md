# glossary/ — словарь компонентов и кросс-ссылки

Один проект / технология / понятие может упоминаться в нескольких разделах монорепозитория. Этот глоссарий собирает все упоминания в одном месте, чтобы быстро находить все ссылки на конкретный компонент.

## Файлы

| Файл | О чём |
|---|---|
| [`components-by-name.md`](components-by-name.md) | Алфавитный список всех проектов/технологий/паттернов с обратными ссылками на доки, где они упоминаются |
| [`authors-by-name.md`](authors-by-name.md) | Алфавитный список авторов и их проектов |
| [`concepts.md`](concepts.md) | Ключевые понятия (CardIndex, evidence envelope, hot/slow path, STDP, etc.) |

## Принцип

- Один компонент — одна строка, перечисляются все доки, где он встречается.
- Ссылки относительные, чтобы работали и в репо, и при просмотре через GitHub UI.
