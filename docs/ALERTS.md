# Callout-блоки

Добавлено **60** callout-блоков в документы.

| Тип | Количество | Назначение |
|-----|------------|------------|
| `[!NOTE]` | 0 | Нейтральная заметка |
| `[!TIP]` | 38 | Практический совет |
| `[!WARNING]` | 21 | Предупреждение о риске |
| `[!IMPORTANT]` | 1 | Ключевой документ |

## Пример синтаксиса

```markdown
> [!NOTE]
> Это заметка.

> [!TIP]
> Практический совет.

> [!WARNING]
> Предупреждение.

> [!IMPORTANT]
> Важная информация.
```

_Поддерживается в GitHub Markdown с 2023 года._
