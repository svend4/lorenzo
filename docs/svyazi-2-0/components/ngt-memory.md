## Описание

Персистентная память для LLM‑приложений с ассоциативным графом и миллисекундным retrieval overhead.

## Ключевые компоненты и паттерны

- Cosine similarity + Hebbian graph + hierarchical consolidation
- REST API
- Docker
- 2–3 ms собственных затрат
