## Главный вывод

См. [`signals.md`](signals.md). Коротко: тезис «кодинг исчезнет первым» и одновременный найм 68 Research Engineer — не противоречие; самый быстрорастущий блок — не разработка, а Sales + Applied AI (≈200 ролей).

## Замечание о размере исходника

Исходный MHTML‑файл — самый большой из всех (~7.6 MB), потому что в этом же диалоге обсуждались и другие темы (профиль `svend4`, репозитории `pro2`/`Nautilus`, маппинг профиля на роли Anthropic, GitHub‑токены, дальнейшее развитие проекта и т. д.). В эту папку вынесена только **аналитическая часть про вакансии Anthropic** — она занимает примерно строки 150–240 из расшифровки. Остальной контекст диалога остаётся в исходном MHTML.
