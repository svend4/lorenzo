# mmorpg-for-programmers/ — гипотеза MMORPG-RPG для технарей

Пользователь предложил «полуфантастический полусредневековый» вариант: MMORPG-RPG переделанная под программистов / технарей. Программирование, управление проектом, code review и коммуникация в виде игровых механик. AI следит за тем, чтобы игровая механика соответствовала **реальной продуктивности**, а не накрутке.
