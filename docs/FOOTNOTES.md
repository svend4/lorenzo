## Как это работает

При первом упоминании термина в файле добавляется `[^term]` — ссылка на сноску в конце документа.

```markdown
Используем MCP[^mcp] для подключения инструментов.

[^mcp]: Model Context Protocol — протокол для AI-инструментов
```
