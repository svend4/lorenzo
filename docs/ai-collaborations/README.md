## Как читать

1. Начните с [`candidates/`](candidates/) — там источник всей цепочки рассуждений (Svyazi → K2-18 / Wikontic / NGT Memory → Hebbian collaboration graph).
2. Затем [`ensembles/`](ensembles/) — девять способов собрать новые свойства из найденных проектов.
3. После — [`continuation/`](continuation/) — десять разделов «как от конструктора перейти к операционной модели Svyazi 2.0».
4. [`fast-tracks/`](fast-tracks/) — компактная сводка пяти ключевых связок, удобно как summary.
