# propose-collaboration

Предложение, к кому обратиться по теме / для конкретной задачи.

## Когда использовать

- "к кому обратиться по теме X"
- "кто специалист по Y"
- "найди мне партнёра для коллаборации по Z"
- "посоветуй автора для совместной работы"

## Шаги

1. **Уточнить тему**
   - Если расплывчато — спросить: какой слой (memory/knowledge/orchestration), задача, ожидаемый формат сотрудничества

2. **Поиск кандидатов**

   ```bash
   # Прямой поиск авторов по теме
   python scripts/improve_faceted_search.py --query "тема" --type people

   # Из CONTACTS.md по слою
   grep -i "слой" docs/CONTACTS.md

   # Из CONTACT_PRIORITY.md — уже отранжировано
   cat docs/CONTACT_PRIORITY.md
   ```

3. **Оценить релевантность каждого**

   Для каждого кандидата:
   - Сколько упоминаний? (`docs/ENTITIES.md`)
   - Какие проекты? (`docs/contacts/<slug>.md`)
   - Уже писали? (`docs/PROGRESS.md`)
   - Каков статус? (см. чекбоксы в контактном файле)

4. **Ранжировать**

   | Параметр | Вес |
   |---|---|
   | Релевантность теме | × 5 |
   | Зрелость проекта | × 3 |
   | Доступность контакта | × 2 |
   | Частота активности | × 2 |
   | Предыдущий контакт | × 1 |

5. **Дать топ-3-5 рекомендаций**

   Шаблон ответа:
   ```markdown
   **Топ-3 кандидата для коллаборации по «X»:**

   1. **[Имя] (@handle)** — [проект]
      - **Почему:** [конкретная синергия]
      - **Слой:** [memory|knowledge|...]
      - **Платформа:** Habr / GitHub
      - **Статус:** не писали / написали / ответил
      - **Файл:** [`docs/contacts/<slug>.md`](...)
      - **Первое сообщение:** черновик уже в файле, можно использовать `propose_outreach`

   2. ...
   ```

6. **Предложить дальнейшие шаги**
   - «Хотите, я подготовлю первое сообщение через `write-contact`?»
   - «Обновить статус → `studied` для топ-3?»
