# find-gaps

Поиск пробелов в базе знаний: упомянутых, но не описанных тем.

## Когда использовать

- "чего не хватает в теме X"
- "какие темы упомянуты, но без документа"
- "где пробелы в базе знаний"
- "что нужно дополнить"

## Шаги

1. **Прочитать `docs/CONTENT_GAPS.md`**
   - Сгенерирован `improve_content_gaps.py`
   - Темы упомянутые ≥3 раз без отдельного документа

2. **Расширенный анализ при необходимости:**
   ```bash
   python scripts/improve_content_gaps.py --min-mentions 3
   ```

3. **Категоризация пробелов:**

   | Тип пробела | Признак | Что делать |
   |---|---|---|
   | Концепт-пробел | термин упомянут, нет в `glossary` | создать `glossary-entry` |
   | Проект-пробел | проект упомянут без карточки | создать `project-component` |
   | Процесс-пробел | процесс описан без формализации | создать `rfc` |
   | Решение-пробел | подразумевается решение без ADR | создать `decision-record` |
   | Контакт-пробел | автор упомянут без файла контактов | `improve_autofill.py` |

4. **Приоритизация:**
   - Сколько раз упомянут? (>10 — критично)
   - В каких разделах? (>3 разделов — пересекающийся)
   - Связан ли с открытыми вопросами в `QUESTIONS.md`?
   - Есть ли action items в `ACTION_ITEMS.md`?

5. **Заполнение:**

   Для каждого приоритетного пробела:

   ```bash
   # Вариант 1: создать через шаблон
   python scripts/improve_template_init.py --type glossary-entry \
     --slug docs/glossary/term-X.md \
     --vars term="X" type=concept

   # Вариант 2: автозаполнение из существующего контекста
   python scripts/improve_subtopic_fill.py --apply --topic X

   # Вариант 3: BM25-цитаты
   python scripts/improve_gap_filler.py --apply --mode cite --top 2
   ```

6. **Отчёт:**

   ```markdown
   # Пробелы в базе знаний

   **Найдено:** 23 пробела
   **Критичных (>10 упоминаний):** 5

   ## Топ-5 для немедленного заполнения

   1. **`Q6-гиперкуб`** — упомянут 18 раз, нет в глоссарии
      - Действие: `improve_template_init.py --type glossary-entry --vars term="Q6-гиперкуб"`

   2. **`Hot path`** — упомянут 14 раз
      - Действие: glossary-entry + ссылка из Yodoca

   ...

   ## Средние (5-10 упоминаний)
   ...
   ```
