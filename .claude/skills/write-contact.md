# Скилл: write-contact

## Когда использовать
Когда пользователь хочет написать сообщение автору OSS-проекта:
- "Напиши письмо автору AgentFS"
- "Составь запрос на коллаборацию для kksudo"
- "Мне нужно написать spbmolot"

## Алгоритм

1. Найди контакты в `docs/CONTACTS.md`
2. Прочитай шаблон `docs/templates/contact-outreach.md`
3. Найди информацию о проекте автора в `docs/05-habr-projects/`
4. Составь персонализированное письмо

## Шаблон письма

```markdown
Здравствуйте, {имя}!

Я {представление} и работаю над проектом Svyazi 2.0 — экосистемой OSS-компонентов 
для AI-систем с памятью и оркестрацией агентов.

Изучил ваш проект {название} ({ссылка}) и считаю, что он идеально дополняет нашу 
архитектуру в части {конкретная роль}.

Предлагаю рассмотреть возможность:
- {конкретное предложение 1}
- {конкретное предложение 2}

Детали архитектурной интеграции описаны в {ссылка на integration-contracts}.

Готов ответить на вопросы. Как вам удобно общаться?

С уважением,
{подпись}
```
