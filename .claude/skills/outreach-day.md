# outreach-day

«День аутрича» — связаться с N приоритетными контактами.

## Когда использовать

- "outreach день"
- "пиши приоритетным контактам"
- "сегодня связываемся с авторами"
- "пройти по списку контактов"

## Композиция

1. [`track-decisions`](track-decisions.md) — какие решения требуют внешней экспертизы
2. [`propose-collaboration`](propose-collaboration.md) — выбрать топ-3-5
3. [`write-contact`](write-contact.md) — написать
4. После отправки → обновить `update_contact_status`

## Шаги

1. **Прочитать приоритеты**
   ```bash
   Read: docs/CONTACT_PRIORITY.md
   ```

2. **Выявить «застрявших»**
   ```bash
   python scripts/improve_contact_status.py --list
   ```
   - В статусе `studied` > 7 дней — почему не написали?
   - В статусе `messaged` > 14 дней без ответа — нужен follow-up?

3. **Для каждого top-N:**

   **3.1 Прочитать контактный файл**
   ```bash
   Read: docs/contacts/<slug>.md
   ```

   **3.2 Прочитать проект автора**
   ```bash
   find docs -iname "*<project>*" | head -3
   ```

   **3.3 Подготовить сообщение**
   - Использовать черновик из секции «Первое сообщение»
   - Адаптировать под текущий контекст:
     - Конкретный вопрос из открытых вопросов
     - Свежая синергия (если что-то добавилось в Lorenzo)
     - Не больше 5-7 предложений

   **3.4 Показать пользователю на одобрение**
   - Если ОК — отправляется (вручную; скилл не отправляет)
   - Если не ОК — итерация

   **3.5 После отправки**
   ```bash
   python scripts/improve_contact_status.py --author <slug> --messaged
   python scripts/improve_contact_status.py --author <slug> --note "Отправил сообщение про X"
   ```

4. **Сводный отчёт по итогам дня**

   ```markdown
   # Outreach: [дата]

   **Связались:** N человек
   **Сообщений отправлено:** M
   **Follow-up'ов:** K

   ## Список
   1. [@author1] — статус: messaged → ждём ответа
   2. ...

   ## Завтра напомнить:
   - [@authorX] — отправили 3 дня назад без ответа

   ## Не получилось связаться:
   - [@authorY] — причина: …
   ```

5. **Обновить PROGRESS.md**
   ```bash
   python scripts/improve_progress_sync.py
   ```

## Антипаттерны
- ❌ Шаблонное сообщение без персонализации
- ❌ Длинное сообщение (> 10 предложений) — никто не читает
- ❌ Без конкретного вопроса — нет повода ответить
- ❌ Выдавать всю Lorenzo архитектуру в первом письме
